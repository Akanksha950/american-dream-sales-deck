# American Dream Mall — Interactive Sales Platform

A high-fidelity, browser-based sales enablement tool designed to replace static PDF pitches. This deck is built for the American Dream commercial team to pitch retail tenants, sponsors, and event producers.

## 🔗 Live Experience
**[VIEW LIVE PROJECT HERE](https://akanksha950.github.io/american-dream-sales-deck/ )**

---

## 🚀 Tech Stack & AI Fluency
* **Architecture:** HTML5, CSS3 (Custom Grid & Flexbox), and Vanilla JavaScript.
* **Design:** Cormorant Garamond (Luxury Serif) & Barlow Condensed (Architectural Sans-serif).
* **Interactivity:** GSAP-inspired scroll-reveal logic and a custom JS-driven cursor ring for a cinematic feel.
* **AI Tools Used:** * **Claude 3.5/GPT-4o:** Used for architectural scaffolding and synthesizing complex regional demographic data.
    * **Midjourney/DALL-E:** Generated high-end background textures and luxury-themed UI placeholders.

## 🎯 Design Rationale
The core objective was to eliminate "pitch friction." 
- **Non-Linear Navigation:** A salesperson can jump to any module (Leasing vs. Events) instantly based on client questions.
- **Visual Hierarchy:** Used a "Dark/Gold" luxury palette to mirror the branding of anchor tenants like Hermès and Gucci.
- **Data Visualization:** Built a custom CSS "Catchment Map" to visually represent the 40M+ person reach within the NYC/Philly/Boston corridor.
